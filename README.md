# Dominoes-Rows
Решение задачи о выкладывании костей домино в ряд на C++ с использованием двусвязного списка. Консольное меню: ручной ввод, случайная генерация, чтение из файла.

Как играть? Правила игры:

  1. Кость домино состоит из двух чисел от 0 до 6.
  2. Ряд считается корректным, если соседние кости соприкасаются одинаковыми числами.
  3. Задача: определить, можно ли выложить заданный набор костей в один ряд (каждая кость используется ровно один раз).
  4. Если можно — вывести один из возможных вариантов расположения.
  5. Кости в наборе уникальны (нет дубликатов даже в перевёрнутом виде).

Как запустить программу?

1 вариант: (работает без сред разработки, запускается на компьютере) Запустить файл cpp.exe

2 вариант: (работает только в средах разработки таких как Clion, VS Community, VS Code и т.д.)

  1. Загрузить файлы с репозитория себе на компьютер
  2. Переместить их в один проект
  3. Запустить программу
